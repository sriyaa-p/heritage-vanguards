Model Context Protocol & Tool Exposure Layer
MCP SERVER IMPLEMENTATION

backend/mcp_server.py exposes the full Heritage Sentinel pipeline as 6 callable tools via the Model Context Protocol — compatible with Gemini ADK, Claude Desktop, and any MCP-compliant agent.

DUAL TRANSPORT MODES

Supports stdio transport for local agent use and SSE (Server-Sent Events) for remote agents connecting over HTTP on port 8001, enabling both local and cloud deployments.

AGENT DECOUPLING

External AI agents interact with Heritage Sentinel without coupling to internal REST API routes. Tool calls are protocol-native, stateless, and schema-validated.

HUMAN-IN-THE-LOOP GATE

record_review_decision enforces that every expert approve / reject / escalate decision is captured before a candidate site changes pipeline status.
